经常使用的分支其实不算太多，不过仓库倒是有好几个。来来去去写命令行或者GUI操作多了觉得有点繁琐，就琢磨来写个脚本做吧。

PS：偷懒是人类进步的动力…

懒得纠结，换shell吧。
